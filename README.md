## Todo List 만들기
2024년 3월 원티드 프리온보딩 FE 챌린지의 사전과제로 진행한 Todo List 입니다. <br />

redux-persist를 사용하여 로컬 스토리지에 데이터를 저장하는 것을 구현하였습니다.
